# Проект "Виджет банковских операций"

## Описание проекта
Проект предназначен для обработки и анализа банковских операций клиента. 
Включает функции фильтрации и сортировки операций.

## Установка
Клонируйте репозиторий:
```bash
git clone https://github.com/MaximxLeon/SkyPro.git
```
## Тестирование

Проект покрыт тестами (pytest).  
Для запуска тестов:

```bash
pytest
```
